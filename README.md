# ❤️ Heart Stroke Prediction using Machine Learning

## 🌐 Live Demo

🔗 **Try the app here:** https://heartstrokepredictor-hf92x9ipudiprrsbmswavh.streamlit.app/


A Machine Learning web application that predicts the likelihood of heart disease based on a patient's medical information. The application is built using **Python**, **Scikit-learn**, and **Streamlit**.

---

## 📌 Project Overview

Heart disease is one of the leading causes of death worldwide. This project aims to assist in early risk assessment by using machine learning algorithms to predict whether a person is at risk of heart disease.

The project includes:
- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Model training and evaluation
- Interactive Streamlit web application

---

## 🚀 Features

- User-friendly Streamlit interface
- Real-time heart disease prediction
- Data preprocessing with StandardScaler
- One-hot encoding of categorical features
- Machine Learning model integration
- Fast and easy-to-use interface

## 📊 Dataset

The project uses a Heart Disease dataset containing patient health information such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- Oldpeak
- ST Slope

---

## 🤖 Machine Learning Models Tested

- Logistic Regression
- K-Nearest Neighbors (KNN) ✅
- Decision Tree
- Naive Bayes
- Support Vector Machine (SVM)

The **KNN model** was selected for deployment based on its performance.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/nickxjarvis1810/HeartStrokePredictor.git
```

Go into the project folder:

```bash
cd HeartStrokePredictor
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app.py
```
